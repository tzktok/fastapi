# 👋 Welcome to BankStmtConverter Documentation

**BankStmtConverter** is a powerful online tool that converts your bank statements into editable formats like **Excel**, **CSV**, or **Google Sheets** — quickly and accurately.

---

### 🚀 Why BankStmtConverter?

Managing bank statements manually can be time-consuming and error-prone. Our **AI-powered platform** simplifies this process by automatically extracting table data from your **PDF** or **image** bank statements.

---

### 🎯 Key Benefits

- ✅ Supports multiple output formats: **Excel**, **CSV**, **HTML**, **XML**, etc.
- ✅ AI-based table detection for highly accurate data extraction
- ✅ Lightning-fast processing with **GPU acceleration**
- ✅ Secure handling of sensitive financial data
- ✅ Download results in **Excel**, **CSV**, or **Google Sheets**

---

### 📊 Use Cases

- 👨‍💼 Accountants and finance teams
- 🧾 Small business owners managing expenses
- 🏦 Individuals organizing personal finances
- 👩‍💻 Developers integrating financial workflows via API
